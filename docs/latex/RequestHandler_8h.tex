\doxysubsection{Detailed Description}
Fichier d\textquotesingle{}en-\/tête pour la gestion des requêtes du serveur. 

Ce fichier contient la fonction {\ttfamily handle\+Server\+Request} qui traite les requêtes du serveur. Il inclut les en-\/têtes nécessaires, définit les constantes pour les types de commandes, et fournit la fonction pour analyser la requête, exécuter la commande correspondante et mettre à jour la réponse.
