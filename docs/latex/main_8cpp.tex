\doxysubsection{Detailed Description}
Fichier principal du programme. 

Ce fichier contient la fonction principale {\ttfamily main} qui lance le serveur et gère les requêtes. Il inclut les en-\/têtes nécessaires, définit les constantes et les variables globales, et crée les objets multimédias et les ajoute à la base de données. Il contient également la fonction {\ttfamily handle\+Server\+Request} qui traite les requêtes du serveur.
